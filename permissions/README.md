# Permission Scripts

<0-iam_betty changes user to 'betty'>
<1-who_am_i prints current user name>
<2-groups prints user groups>
<3-new_owner changes user of file hello to betty>
<4-empty creates new file called empty>
<5-execute grants execute permission to file empty>
<6-multiple_permissions grants specific permissions to user, group, and all users>
<7-everybody grants execute permissions to all users>
<8-James_Bond modifies permissions to 007>
<9-John_Doe modifies permissions to 753>
<10-mirror_permissions copies permissions from another file>
<11-directories_permissions grants access to all subdirectories>
<12-directory_permissions creates my_dir and sets permissions>
<13-change_group changes group owner>
<14-chang_owner_and_group changes file owner and group on all files recursivly>
<15-symbolic_link_permissions modifies permissions on symbolic link only>
<16-if_only modifies owner from one specific user to another>
